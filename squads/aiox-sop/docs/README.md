# SOP Factory Docs

This directory is reserved for squad-local reports such as optimization plans, validation outputs, and post-run evidence generated by maintenance workflows.

Canonical share-safe markdown SOPs live in root `docs/sops/`.

Portable operational source of truth lives at `docs/squad/aiox-sop/operational-projection.yaml`.

Business-aware workspace context is loaded on demand via
`scripts/load-workspace-context.cjs`; it is not the default mode for the squad.
