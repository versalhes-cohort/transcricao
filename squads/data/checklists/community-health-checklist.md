# Community Health Checklist
Clone: @david-spinks (Tier 1)

Validate community strategy and health before reporting or changes.

## 1. SPACES Model Assessment

### 1.1 Support
- [ ] Members can get help from other members
- [ ] Response time to questions tracked
- [ ] Community-answered % vs staff-answered tracked
- [ ] Knowledge base/FAQ contributed by community

### 1.2 Product
- [ ] Feedback channel exists and is active
- [ ] Feature requests tracked and visible
- [ ] Beta testing group identified
- [ ] Product team engages with community input

### 1.3 Acquisition
- [ ] Referral program exists
- [ ] Member-generated content drives SEO
- [ ] Word-of-mouth tracked
- [ ] Success stories shared by members

### 1.4 Contribution
- [ ] User-generated content encouraged
- [ ] Contributors recognized/rewarded
- [ ] Clear contribution guidelines exist
- [ ] Quality control for UGC in place

### 1.5 Engagement
- [ ] Regular events/programming scheduled
- [ ] Multiple engagement formats available
- [ ] Participation tracked per member
- [ ] Engagement trends monitored

### 1.6 Success
- [ ] Member outcomes tracked
- [ ] Success stories documented
- [ ] NPS/satisfaction measured
- [ ] Churn correlated with community engagement

## 2. Core Metrics Health

### 2.1 Activity Metrics
- [ ] MAU (Monthly Active Users) tracked
- [ ] DAU (Daily Active Users) tracked
- [ ] Stickiness ratio calculated (DAU/MAU)
- [ ] Trend direction documented

### 2.2 Engagement Distribution
- [ ] % of members posting tracked
- [ ] Lurker % understood and acceptable
- [ ] Power users identified
- [ ] Engagement funnel mapped

### 2.3 Content Metrics
- [ ] Posts per day/week tracked
- [ ] Comments/reactions tracked
- [ ] Content quality assessed
- [ ] Most engaging content types identified

## 3. Three-Level Strategy Check

### 3.1 Level 1: Tactics
- [ ] Current programs documented
- [ ] Event calendar exists
- [ ] Content calendar exists
- [ ] Resource allocation clear

### 3.2 Level 2: Strategy
- [ ] Business objectives defined
- [ ] Community tied to business goals
- [ ] ROI measurement in place
- [ ] Success metrics aligned with business

### 3.3 Level 3: Culture
- [ ] Community values defined
- [ ] Moderation guidelines clear
- [ ] Culture actively cultivated
- [ ] Toxic behavior addressed

## 4. Health Signals

### 4.1 Positive Signals
- [ ] New member retention stable/improving
- [ ] Organic conversations happening
- [ ] Members helping members
- [ ] Advocacy/referrals increasing

### 4.2 Warning Signs to Monitor
- [ ] Engagement declining over 3+ months
- [ ] Same small group doing all posting
- [ ] Complaints/negativity increasing
- [ ] Staff-to-community ratio too high

## 5. Operational Health

### 5.1 Team & Resources
- [ ] Community team adequately staffed
- [ ] Tools/platform adequate
- [ ] Budget sufficient for programming
- [ ] Escalation path for issues clear

### 5.2 Documentation
- [ ] Community guidelines published
- [ ] Onboarding flow documented
- [ ] Moderation playbook exists
- [ ] Emergency response plan ready
