## Pre-Flight Checklist

Before building any upsell page, verify these prerequisites:

### Technical Prerequisites
- [ ] **One-Click Payment Configured** - Token storage from main purchase working
- [ ] **Timer System Ready** - Countdown mechanism with sound capability
- [ ] **Exit Intent Detection Active** - Mouse/back button triggers configured
- [ ] **Video Hosting Ready** - Autoplay and completion tracking enabled
- [ ] **Mobile Responsive Template** - Fixed footer buttons working

### Content Prerequisites
- [ ] **Upsell Offer Defined** - Clear complementary product/service
- [ ] **Pricing Strategy Set** - 30-50% of main product value
- [ ] **Downsell Variant Ready** - 15-25% of main product value
- [ ] **Testimonials Collected** - 2-3 upsell-specific testimonials
- [ ] **VSL Script Written** - 2-4 minute structure complete

---

## SECTION A: O Topo Critico (Above the Fold)

### Element 1: Headline de Continuidade
- [ ] **Maintains "yes" momentum** - Feels like continuation, not new pitch
- [ ] **Validates previous purchase** - Confirms their decision was good
- [ ] **Includes personalization** - Uses customer name if available
- [ ] **Creates anticipation** - Hints at additional value

**Validation Checks:**
- [ ] Does NOT question previous purchase
- [ ] Does NOT create new anxiety
- [ ] Does NOT imply main product is insufficient

**Template Options Used:**
- [ ] "ESPERE! Sua compra foi confirmada, mas..."
- [ ] "[Nome], voce desbloqueou uma oferta exclusiva"
- [ ] "Parabens! Antes de acessar seu [produto], veja isso..."
- [ ] Custom (must pass validation checks)

---

### Element 2: Subheadline de Complementaridade
- [ ] **Positions as multiplier** - Upsell accelerates, doesn't replace
- [ ] **States specific result** - Clear additional outcome
- [ ] **Includes benefit multiplier** - "3x faster" or similar
- [ ] **Contains time limit** - "Oferta valida apenas agora"

**Formula Applied:**
```
"Adicione [RESULTADO ESPECIFICO] ao que voce acabou de comprar
e [BENEFICIO MULTIPLICADOR] - Oferta valida apenas agora"
```

---

### Element 3: Barra de Progresso Psicologica
- [ ] **3 steps visible** - Clear progression
- [ ] **Step 1 has green checkmark** - Reinforces positive decision
- [ ] **Step 2 shows "Voce esta aqui"** - Creates incomplete mission feeling
- [ ] **Step 3 shows final destination** - Area de membros/access

**Visual Structure:**
```
[✓] Passo 1: Programa Principal Garantido
[?] Passo 2: Acelerar Resultados (Voce esta aqui)
[ ] Passo 3: Acessar Area de Membros
```
- [ ] Checkmark verde visible
- [ ] Current step highlighted
- [ ] Incomplete step grayed

---

### Element 4: Timer de Decisao
- [ ] **Duration set** - Maximum 10 minutes (not 24h)
- [ ] **Message displayed** - "Oferta expira quando sair desta pagina"
- [ ] **Sound enabled** - Tique-taque nos ultimos 60 segundos
- [ ] **Expiry behavior** - Redirects to thank you page

**Timer Configuration:**
- [ ] Timer visible above fold
- [ ] Countdown active on page load
- [ ] Sound triggers at 60s remaining
- [ ] Page redirects at 0

---

## SECTION B: O Corpo Persuasivo

### Element 5: Video Upsell Curto
- [ ] **Duration** - 2-4 minutes maximum (NEVER exceed)
- [ ] **Opens with congratulations** - Validates purchase (0-10s)
- [ ] **Reveals gap** - What upsell addresses (10-30s)
- [ ] **Demonstrates result** - Proof of additional value (30-90s)
- [ ] **Quick value stack** - Components listed (90-180s)
- [ ] **Urgent CTA** - Clear action (180-240s)

**Opening Script Matches:**
```
"Parabens! Voce acabou de tomar a melhor decisao para [resultado].
Mas descobri que alunos que tambem pegam [upsell] conseguem
[resultado 3x maior]. Deixe-me mostrar rapidamente..."
```
- [ ] Script follows template
- [ ] Tone is celebratory, not salesy
- [ ] No new anxiety created

---

### Element 6: Grafico "Com vs Sem"
- [ ] **Side-by-side format** - Clear visual comparison
- [ ] **5 comparison points** - Specific benefits listed
- [ ] **Checkmarks/X marks** - Visual yes/no indicators
- [ ] **Velocity multiplier footer** - "1x vs 3-5x" bottom line

**Comparison Points Include:**
- [ ] Time to result
- [ ] Support level
- [ ] Resources/templates included
- [ ] Implementation method
- [ ] Learning curve/roadmap

---

### Element 7: Mini Stack de Valor
- [ ] **Maximum 3 components** - Simplicidade acelera decisao
- [ ] **Each has dollar value** - Clear value assignment
- [ ] **Total calculated** - Sum displayed
- [ ] **Today's price shown** - Significant discount from total
- [ ] **5-10x ratio validated** - Perceived value >= 5x price

**Value Stack Structure:**
```
COMPONENTE PRINCIPAL .............. Valor R$ [X]
BONUS RAPIDO #1 .................. Valor R$ [Y]
BONUS RAPIDO #2 .................. Valor R$ [Z]
                    TOTAL: R$ [X+Y+Z]
                    HOJE APENAS: R$ [price]
```

**Ratio Check:**
- [ ] Total value / Today's price >= 5x (minimum)
- [ ] Total value / Today's price >= 10x (optimal)

---

### Element 8: Prova Social Especifica do Upsell
- [ ] **Quantity** - 2-3 testimonials only
- [ ] **Upsell-specific** - About the upsell, NOT main product
- [ ] **Short format** - Quick to read
- [ ] **Photos included** - Real faces
- [ ] **Specific results** - Quantified outcomes

**Example Format:**
```
"93% dos alunos que pegaram o [Upsell] dizem que
foi a melhor decisao que tomaram..."

[Foto] "O [upsell] cortou meu tempo de implementacao
em 75%. Resultados na primeira semana!" - Nome
```

---

### Element 9: Garantia Empilhada
- [ ] **Original guarantee referenced** - "Sua garantia original CONTINUA VALENDO"
- [ ] **Additional guarantee added** - Extra protection for upsell
- [ ] **Specific outcome promised** - "acelerar resultados em pelo menos 2x"
- [ ] **Clear refund terms** - "devolvemos 100% do valor do upgrade"
- [ ] **Keep everything clause** - "voce fica com tudo"

---

## SECTION C: Elementos de Conversao Rapida

### Element 10: Dois Botoes Apenas
- [ ] **Binary choice** - Only Yes and No options
- [ ] **Yes button** - Verde, Grande, Primary position
- [ ] **No button** - Cinza, Menor, Secondary position
- [ ] **Outcome-focused copy** - Results, not features

**Button Copy:**
- [ ] YES: "SIM! Quero Resultados 3x Mais Rapidos →"
- [ ] NO: "Nao, prefiro ir mais devagar"

**Mobile Specific:**
- [ ] Buttons fixed at bottom of viewport
- [ ] Tap targets appropriately sized
- [ ] Always visible during scroll

---

### Element 11: Micro-Comparacao de Preco
- [ ] **Main price referenced** - "Voce acabou de investir R$ [main]"
- [ ] **Percentage shown** - "menos de 30% extra"
- [ ] **Multiplier stated** - "multiplica resultados por 3"
- [ ] **Metaphor included** - "E como comprar um carro e deixar de pegar o seguro"

---

### Element 12: FAQ Ultra-Especifica
- [ ] **Exactly 3 questions** - No more, no less
- [ ] **Addresses timing objection** - "Preciso disso agora ou posso pegar depois?"
- [ ] **Addresses value objection** - "Isso nao deveria estar incluido?"
- [ ] **Addresses risk objection** - "E se eu me arrepender?"

**Answers Match Templates:**
- [ ] Q1 Answer mentions exclusive pricing
- [ ] Q2 Answer differentiates main vs upsell purpose
- [ ] Q3 Answer references guarantee + extra days

---

### Element 13: Reassurance Bar
- [ ] **Security indicator** - "🔒 Compra Segura"
- [ ] **Same card mention** - "Mesmo cartao ja cadastrado"
- [ ] **Guarantee reference** - "Garantia Total"
- [ ] **Access timing** - "Acesso Imediato"

---

### Element 14: Exit Intent Pop-up Downsell
- [ ] **Trigger configured** - Mouse to X or back button
- [ ] **Headline** - "ULTIMA CHANCE!"
- [ ] **Empathy statement** - "Ok, entendo que R$ [price] pode ser demais agora..."
- [ ] **Payment plan offer** - "3x de R$ [split price]"
- [ ] **Two buttons** - Accept and decline

**Modal Contains:**
```
"ULTIMA CHANCE!
Ok, entendo que R$ 497 pode ser demais agora...
Que tal levar o mesmo upgrade por 3x de R$ 197?"
[SIM, QUERO PARCELADO] [Nao, obrigado]
```

## SECTION D: Sequencia de Multiplos Upsells

### Element 15: Logica de Progressao
**Upsell #1 (Complemento Direto):**
- [ ] Pricing: 30-50% of main product value
- [ ] Purpose: Accelerates main result
- [ ] Target conversion: 30%
- [ ] If accepted → Upsell #3
- [ ] If rejected → Upsell #2

**Upsell #2 (Versao Lite):**
- [ ] Pricing: 15-25% of main product value
- [ ] Purpose: Lower barrier entry
- [ ] Target conversion: 20%
- [ ] If accepted → Thank You Page
- [ ] If rejected → Thank You Page

**Upsell #3 (Complemento Premium):**
- [ ] Pricing: 50-100% of main product value
- [ ] Purpose: Maximum value for committed buyers
- [ ] Target conversion: 15% of Upsell #1 buyers
- [ ] If accepted → Thank You Page
- [ ] If rejected → Thank You Page

---

### Element 16: Thank You Page Optimization
- [ ] **Confirmation visual** - All purchases summarized
- [ ] **Proximos passos** - Numbered next steps
- [ ] **Direct link** - Area de membros access
- [ ] **Onboarding video** - 2 minutes max
- [ ] **Referral offer** - "Refer a Friend" with incentive

---

### Element 17: One-Click Upsell Tecnico
- [ ] **Token stored** - From first purchase
- [ ] **Single click processing** - No re-entry of data
- [ ] **Instant confirmation** - <3 second processing
- [ ] **Fallback available** - Manual form if token fails
- [ ] **Tested before launch** - Verified working

**Critical Rule:**
> "Configure o one-click upsell ANTES de criar a pagina.
> A friccao tecnica mata mais upsells que copy ruim."

## Performance Validation

### Target Metrics
- [ ] **Upsell #1 Take Rate** - Target: >25%
- [ ] **AOV Multiplier** - Target: 1.8-2.5x main product
- [ ] **Time on Page** - Target: <90 seconds (quick decision)
- [ ] **Exit Intent Conversion** - Target: >10% of triggers
- [ ] **Technical Success Rate** - Target: >99%

### Launch Checklist
- [ ] All 17 elements implemented
- [ ] All section A elements above fold
- [ ] Mobile experience tested
- [ ] One-click payment verified
- [ ] Timer and sound working
- [ ] Exit intent triggering correctly
- [ ] Analytics tracking active
- [ ] A/B testing ready (if applicable)

## Quick Reference: What NEVER To Do

### Headlines
- [ ] Never question the purchase ("Tem certeza?")
- [ ] Never create new anxiety ("Voce esta perdendo...")
- [ ] Never imply insufficiency ("O programa sozinho nao e suficiente...")

### Value Stack
- [ ] Never exceed 3 components
- [ ] Never have <5x perceived value ratio

### VSL
- [ ] Never exceed 4 minutes
- [ ] Never skip the congratulations opening

### FAQ
- [ ] Never have more than 3 questions
- [ ] Never have fewer than 3 questions

### Timing
- [ ] Never set timer >10 minutes
- [ ] Never allow page access after timer expires

---

## Troubleshooting Quick Guide

| Symptom | First Check | Likely Cause |
|---------|-------------|--------------|
| Take rate <15% | One-click working? | Technical friction |
| Time on page >120s | VSL length? | Content fatigue |
| Low exit intent conversion | Downsell offer value? | Weak alternative |
| High page abandonment | Headline continuity? | Momentum broken |
| Low VSL completion | First 30 seconds? | Weak hook |
