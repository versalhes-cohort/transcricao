## Pre-Flight Checklist

Before launching a Fast Cash Play, verify:

### List Readiness
- [ ] Customer database accessible
- [ ] Existing customers identified and segmented
- [ ] Past customers (in good standing) identified
- [ ] Engaged leads with explicit permission identified
- [ ] Cold leads (no contact 6+ months) EXCLUDED
- [ ] Contact information verified (email, phone)
- [ ] List count calculated for 5-10% target

### Offer Readiness
- [ ] Ultra-premium offer designed
- [ ] 1-3 core components defined
- [ ] 3 bonuses separated and ready to layer
- [ ] Bonus 1 (good) ready for Day 5/Day 2 release
- [ ] Bonus 2 (better) ready for Day 2/6h release
- [ ] Bonus 3 (best - "worth more than entire price") ready for 24h/cart open
- [ ] Price set at 10x-50x current average cart value
- [ ] Quantity cap set at 5-10% of customer base

### Fulfillment Readiness
- [ ] Delivery capacity verified for all potential buyers
- [ ] Schedule blocked for premium service delivery
- [ ] Team briefed on premium expectations
- [ ] "You. Will. Work. More. For. These. Customers" understood

### Materials Readiness
- [ ] All emails written and loaded in CRM
- [ ] All texts written and scheduled
- [ ] CTA consistent across all messages
- [ ] Price indication included where appropriate
- [ ] Deadline (Saturday midnight) set clearly
- [ ] Booking/checkout links tested

### Sales Readiness (Consult Path)
- [ ] Calendar open for consults over 7-day window
- [ ] Sales scripts reviewed
- [ ] Card-on-file charging process ready
- [ ] "I'm in" reply handling process defined

### Tech Readiness (Auto Checkout Path)
- [ ] Checkout page built and tested
- [ ] Payment processing verified
- [ ] Cart open time scheduled
- [ ] Inventory/spot limiting configured

## Phase I: Preparation Checklist

### Step 1: Assemble Target List
- [ ] Query CRM for existing customers
- [ ] Query CRM for past customers in good standing
- [ ] Query CRM for engaged leads with permission
- [ ] Exclude leads with no contact in 6+ months
- [ ] Verify contact info for all included
- [ ] Calculate total list size
- [ ] Set quantity cap (5-10% of list or fulfillment limit)

### Step 2: Design Ultra-Premium Offer
**Core Components (pick 1-3):**
- [ ] **Attention**: 1-on-1 or many-on-1 from senior person
- [ ] **Personalization**: Customized/tailored version
- [ ] **Convenience**: 24/7 support, extended hours, exclusive times
- [ ] **Status**: Special perks, badges, recognition
- [ ] **Duration**: 6 months to 1 year commitment
- [ ] **Speed**: Priority, <10 min response promise
- [ ] **Experience**: White glove, concierge, nice dinners
- [ ] **Access**: In-person, retreats, behind the scenes
- [ ] **Network**: Connection with other top customers
- [ ] **Secrets**: Trade secrets, vetted vendors, black book

**Bonus Structure:**
- [ ] Bonus 1 defined and benefits documented
- [ ] Bonus 2 defined and benefits documented
- [ ] Bonus 3 defined (must be worth more than price tag)
- [ ] All bonuses available to all buyers regardless of timing

### Step 3: Set Premium Price
- [ ] Current average cart value identified: $________
- [ ] 10x multiplier = $________
- [ ] 50x multiplier = $________
- [ ] Final price set: $________
- [ ] Price shocks as much as value shocks: VERIFIED

### Step 4: Set Quantity Cap
- [ ] Customer base count: ________
- [ ] 5% of customers: ________
- [ ] 10% of customers: ________
- [ ] Fulfillment capacity: ________
- [ ] Final cap (lowest of above): ________ spots

### Step 5: Choose Selling Method
- [ ] **Consult Path** (7 days) - IF:
  - [ ] High price point
  - [ ] Limited sales staff
  - [ ] Need to spread appointments
- [ ] **Auto Checkout Path** (5 days) - IF:
  - [ ] Very strong brand
  - [ ] Can handle stampede
  - [ ] Low-friction purchase

### Step 6: Prep All Materials
- [ ] All emails written (see templates)
- [ ] All texts written (see schedule)
- [ ] Loaded into CRM/scheduler
- [ ] Send times configured
- [ ] Test messages sent
- [ ] Links verified working

## Phase II: Execution - Push to Consult (7 Days)

### Day 7 (Sunday): Primer
- [ ] **TEXT SENT**: "Hey [CLIENT NAME] would it be cool if I got you [TO THEIR GOAL] and saved you a ton of money? Give me a thumbs up if that sounds cool. I'm sending you something you're gonna like tomorrow."
- [ ] Responses monitored
- [ ] Hot leads noted

### Day 6 (Monday): Announcement
- [ ] **EMAIL SENT**: "We need to talk" - Core offer, price indication, deadline, CTA to book consult
- [ ] **TEXT AM SENT**: "Hey, sooo I just sent you an email!! Did you get it?"
- [ ] **TEXT PM SENT**: "That email about our special offer has my inbox blowing up. Click >>LINK<< to set a time."
- [ ] Consult bookings tracked
- [ ] Early sales recorded

### Day 5 (Tuesday): Recap + Bonus 1
- [ ] **EMAIL SENT**: "best. deal. ever." - Recap all benefits for people who missed it
- [ ] **BONUS 1 RELEASED**: ________________ (Status type)
- [ ] **TEXT SENT**: Meme + link
- [ ] Sales count updated: ____/____

### Day 4 (Wednesday): Mid-week Push
- [ ] **TEXT SENT**: "This offer won't happen ever again. Click >>LINK<< to talk about how it can help you."
- [ ] Sales velocity assessed
- [ ] IF <50% sold: Consider adding value to offer

### Day 3 (Thursday): Sales Update
- [ ] **EMAIL SENT**: "how to [get dream result] and save money doing it"
- [ ] Include actual numbers: "Over half the spots are gone. [X] people have already..."
- [ ] Remind: first-come-first-serve
- [ ] Sales count updated: ____/____

### Day 2 (Friday): Scarcity + Bonus 2
- [ ] **TEXT SENT**: "[CLIENT NAME] We're down to our last couple of spots..."
- [ ] **EMAIL SENT**: "2 spots left + game changer"
- [ ] **BONUS 2 RELEASED**: ________________ (Secrets type)
- [ ] "GAME CHANGER I WAS SAVING UNTIL NOW" messaging used
- [ ] Sales count updated: ____/____

### Day 1 AM (Saturday): Warning + Best Bonus
- [ ] **EMAIL SENT**: "Saving $XXX ends tonight. Don't kick yourself tomorrow."
- [ ] **BONUS 3 RELEASED**: ________________ (Access type - worth more than price)
- [ ] **TEXT SENT**: Reminder with CTA
- [ ] Deadline reinforced: "expires tonight at 7pm"
- [ ] Sales count updated: ____/____

### Day 1 PM (Saturday): Sold Out
- [ ] **IF SOLD OUT**: Send sold out text
- [ ] **IF NOT SOLD OUT**: Consider final push or accept current sales
- [ ] "The program sold out!" notification sent
- [ ] Final sales recorded: ____/____
- [ ] Waitlist compiled (for back-outs)

### Post-Campaign: Back-Out Handling
- [ ] **IF CUSTOMER BACKS OUT**: Send "one spot left" notification
- [ ] Waitlist contacted first
- [ ] Back-out spot filled
- [ ] "RE: your goals we set" email template used

## Troubleshooting Quick Reference

### Not Selling Fast Enough
- [ ] Check open rates (list engagement issue?)
- [ ] Check click rates (offer interest issue?)
- [ ] Check booking/purchase rates (value/price issue?)
- [ ] **ACTION**: Add value to offer before remaining sends
- [ ] **QUOTE**: "Consider adding things to the offer to make it more compelling"

### High Opens, Low Conversions
- [ ] Review offer clarity
- [ ] Review price vs. value presentation
- [ ] Check CTA friction (too many steps?)
- [ ] **ACTION**: Simplify purchase process

### Customer Backing Out
- [ ] Identify reason
- [ ] Improve onboarding/reassurance
- [ ] Send back-out notification to waitlist
- [ ] **ACTION**: Fill spot immediately

### List Fatigue (for future campaigns)
- [ ] Extend cooldown to 6 months
- [ ] Increase value content between campaigns
- [ ] Ensure scarcity was real (don't fake sold-out)
- [ ] **QUOTE**: "If you feel you need to give it longer than twelve weeks to cool off between promotions, run it twice a year instead"
