## Pre-Flight Checklist

Before starting content creation operations, verify:

### Infrastructure
- [ ] Note-taking system always accessible (phone/watch/notepad)
- [ ] Recording equipment operational (camera, mic, screen capture)
- [ ] Platform accounts created and verified
- [ ] Content calendar system ready
- [ ] Analytics tracking configured per platform
- [ ] Lead capture system ready (if monetizing)

### Knowledge Foundation
- [ ] Five topic categories understood (Distant Past, Recent Past, Present, Trends, Fabricated)
- [ ] Seven headline components memorized (Recency, Relevance, Celebrity, Proximity, Conflict, Unusual, Ongoing)
- [ ] Three retention structures mastered (Lists, Steps, Stories)
- [ ] Give:Ask ratio principles understood

### Content Bank
- [ ] Distant Past lessons documented
- [ ] Recent Past experiences catalogued (last week's calendar reviewed)
- [ ] Idea capture habit established
- [ ] Trending topics monitoring active
- [ ] Fabricated experience ideas listed

## Phase I: Hook Checklist

### Topic Selection
**Passado Distante (Distant Past):**
- [ ] Important life lessons identified
- [ ] Connected to product/service value
- [ ] "Story without the scar" approach used
- [ ] Relatable to target audience

**Passado Recente (Recent Past):**
- [ ] Last week's calendar reviewed
- [ ] Meeting insights extracted
- [ ] Conversation lessons captured
- [ ] Client success stories documented
- [ ] Testimonials ready for content

**Presente (Present):**
- [ ] Ideas captured as they come
- [ ] Recording method within arm's reach
- [ ] High-performing ideas marked for expansion
- [ ] Public note-taking active (if applicable)

**Tendencias (Trends):**
- [ ] Current trends identified
- [ ] Personal perspective/experience applied
- [ ] Relevance to expertise established
- [ ] Timing aligned (while still trending)

**Fabricado (Fabricated):**
- [ ] Experience idea documented
- [ ] Effort vs return assessed
- [ ] Content potential evaluated
- [ ] Execution plan ready

### Headline Crafting
**Include at least TWO of these components:**

| Component | Check | Application |
|-----------|-------|-------------|
| Recencia (Recency) | [ ] | Most recent possible |
| Relevancia (Relevance) | [ ] | Personally significant to audience |
| Celebridade (Celebrity) | [ ] | Including prominent people |
| Proximidade (Proximity) | [ ] | Close to home geographically |
| Conflito (Conflict) | [ ] | Opposing ideas/people/nature |
| Incomum (Unusual) | [ ] | Strange, unique, rare, bizarre |
| Em andamento (Ongoing) | [ ] | Stories still in progress |

**Headline Validation:**
- [ ] Gives clear reason to consume
- [ ] Stands out from alternatives
- [ ] Promises specific value
- [ ] Appropriate length for platform

### Format Matching
- [ ] Target platform identified
- [ ] Top-performing content in niche studied
- [ ] Visual style matched to platform norms
- [ ] Length appropriate for platform
- [ ] Format familiar to audience expectations

## Phase II: Retain Checklist

### Lists
- [ ] Items follow theme
- [ ] Number stated in headline or first seconds
- [ ] Each item provides value
- [ ] Order logical (even if not required)
- [ ] Embedded curiosity: "they want to know all items"

**List Templates:**
- "X Ways I [achieved result]"
- "X [things] That [outcome]"
- "X Mistakes [avatar] Make"
- "X Lessons From [experience]"

### Steps
- [ ] Actions in correct order
- [ ] Clear outcome when completed
- [ ] First steps valuable and clear
- [ ] Each step builds on previous
- [ ] Completion feels achievable

**Steps Templates:**
- "X Steps to [achieve result]"
- "How I [do thing] in X Steps"
- "The [Routine/Process] That [outcome]"

### Stories
- [ ] Clear beginning, middle, end
- [ ] Lesson or conclusion included
- [ ] Relatable characters/situations
- [ ] Tension or conflict present
- [ ] Resolution satisfies curiosity

**Story Types:**
- [ ] Things that happened (real experiences)
- [ ] Things that could happen (scenarios)
- [ ] Things that will never happen (imagination)

### Interweaving
For complex content:
- [ ] Lists within steps considered
- [ ] Stories for each list item considered
- [ ] Stories to reinforce step value
- [ ] Multiple storylines if appropriate
- [ ] Creativity maximized

### Short vs Long Form
- [ ] Length matches platform norms
- [ ] Value per second maintained throughout
- [ ] If long: multiple content units linked together
- [ ] If short: single focused content unit
- [ ] Started smaller, building to longer over time

---

## Phase III: Reward Checklist

### Value Per Second
- [ ] Rewards happen frequently throughout
- [ ] No boring sections (cut or improve)
- [ ] Pacing maintains engagement
- [ ] Information density appropriate
- [ ] "No such thing as too long, only too boring" principle applied

### Promise Fulfillment
- [ ] Hook promise clearly identified
- [ ] Content delivers exactly what was promised
- [ ] Quantity matches (if promised 7, give 7)
- [ ] Quality matches expectations
- [ ] Audience can actually USE the information

**Failure Prevention:**
- [ ] NOT giving fewer items than promised
- [ ] NOT giving bad/known items
- [ ] NOT speaking to wrong audience
- [ ] NOT making unusable promises

### Expectation Exceeding
- [ ] More value than expected delivered
- [ ] Bonus insights included
- [ ] Actionable takeaways clear
- [ ] Reason to share created
- [ ] Reason to come back created

### Success Validation
- [ ] Audience growth tracked
- [ ] If growing: content is rewarding
- [ ] If stagnant: content needs improvement
- [ ] If shrinking: major quality issue to address
- [ ] Continuous improvement mindset active

## Phase IV: Monetization Checklist

### Give:Ask Ratio Assessment

| Platform Type | Minimum Ratio | Target Ratio |
|---------------|---------------|--------------|
| Mature (TV, FB) | 3.5:1 | 4:1 |
| Growing | 10:1 | Give until they ask |
| Building Audience | N/A | Pure give |

**Current Ratio Calculation:**
- [ ] Total "give" content counted
- [ ] Total "ask" content counted
- [ ] Ratio calculated
- [ ] Ratio exceeds minimum
- [ ] Audience growth not slowing after asks

### Integration Method Selection

**Integrated (for long-form):**
- [ ] Give:ask ratio maintained within piece
- [ ] Offers placed after valuable moments
- [ ] Offers placed at content end
- [ ] Not over-asking (friend's podcast warning)
- [ ] Growth not slowing after integration

**Example Check:**
- [ ] 60min podcast = max 1.5min total asks
- [ ] 3x 30-second ads = acceptable

**Intermittent (for short-form):**
- [ ] 10 give posts before 1 ask post
- [ ] Ask posts clearly valuable too
- [ ] Pattern sustainable
- [ ] Growth not slowing after asks

### Offer Type Selection

**Lead Magnet (Lower Risk):**
- [ ] Lead magnet created
- [ ] Relevant to content topic
- [ ] Clear value proposition
- [ ] Easy opt-in process
- [ ] Thank you page has main offer

**Lead Magnet Script:**
```
"I have [X] more tips that helped me [result].
Go to [destination] to get a nice visual of them."
```
- [ ] Script adapted to your offer

**Main Offer (Direct Path):**
- [ ] Offer clearly defined
- [ ] Specific avatar called out
- [ ] Dream result stated
- [ ] Time delay mentioned
- [ ] Effort/sacrifice removed
- [ ] Guarantee included
- [ ] Clear CTA

**Main Offer Template:**
```
"I'm looking for [number] [specific avatar] to help achieve
[dream result] in [time delay]. Best part: you don't need
[effort/sacrifice]. If you don't get [dream result]:
1) I'll refund your money
2) I'll work with you until you do.
If that sounds fair, [CTA: DM/schedule/comment/reply]..."
```
- [ ] Template customized to your offer

### Monetization Timing
- [ ] Patience vs urgency assessed
- [ ] Growth priority vs revenue priority decided
- [ ] "Give until they ask" considered first
- [ ] Business needs balanced with growth potential

## Phase V: Scaling Checklist

### Current State Assessment
- [ ] Primary platform identified
- [ ] Current posting frequency documented
- [ ] Current growth rate calculated
- [ ] Quality level assessed
- [ ] Resource capacity evaluated

### Strategy Selection

**Depth-First (Profundidade depois largura):**
Best for: Starting out, limited resources

- [ ] Step 1: Post on one relevant platform
- [ ] Step 2: Post regularly on that platform
- [ ] Step 3: Maximize quality AND quantity
  - Short-form: up to 10x/day
  - Long-form: 5x/week
- [ ] Step 4: Add another platform (maintain first)
- [ ] Step 5: Repeat until all platforms maximized

**Breadth-First (Largura depois profundidade):**
Best for: Existing business, resources available

- [ ] Step 1: Post on one relevant platform
- [ ] Step 2: Post regularly
- [ ] Step 3: Move to next platform (maintain previous)
- [ ] Step 4: Continue until on all relevant platforms
- [ ] Step 5: Maximize all platforms together

### Volume Scaling
- [ ] Current posting frequency documented
- [ ] Target posting frequency set
- [ ] Quality maintained at higher volume
- [ ] 10x content = 10x growth principle remembered
- [ ] Sustainable pace established

### Repurposing Strategy
- [ ] Long-form content identified for repurposing
- [ ] Short-form formats mapped
- [ ] Platform-specific adaptations planned
- [ ] Quality maintained across formats
- [ ] Efficiency gained without quality loss

## Voice & Authenticity Checklist

### "How I" vs "How To"
- [ ] Speaking from experience, not prescription
- [ ] "I do X" instead of "You should do X"
- [ ] "My favorite ways" instead of "The best ways"
- [ ] Experience-based claims only
- [ ] Unquestionable authenticity

**Good Examples:**
- "I make my oatmeal this way"
- "How I Built My 7-Figure Agency"
- "My favorite way to generate leads"

**Bad Examples to Avoid:**
- "You should make your oatmeal this way"
- "How to Build a 7-Figure Agency"
- "This is the best way to generate leads"

### Differentiation
- [ ] Only I can say this
- [ ] Unique experience included
- [ ] Personal perspective clear
- [ ] Not generic advice anyone could give

### Repetition Permission
- [ ] Repeating important messages
- [ ] Not worried about "already said it"
- [ ] Remembering: 1 in 5 haven't seen it
- [ ] Bored with content before audience is

### Seven Lessons Applied

| # | Lesson | Check |
|---|--------|-------|
| 1 | How I vs How To | [ ] Speaking from experience |
| 2 | Reminded vs Taught | [ ] Repeating important things |
| 3 | Puddles to Oceans | [ ] Starting narrow, expanding over time |
| 4 | Content = Sales Tools | [ ] Greatest hits list for sales team |
| 5 | Free Retains Paid | [ ] Free content is valuable |
| 6 | Higher Standards | [ ] Making content better, not shorter |
| 7 | Avoid Pre-Scheduling | [ ] Manual posting when possible |

## Troubleshooting Quick Reference

### Audience Not Growing
- [ ] Check share rate (reward quality indicator)
- [ ] Review headline components
- [ ] Compare posting frequency to benchmarks
- [ ] Analyze platform-specific metrics
- [ ] Remember: "If not growing, content not good enough"

### High Views, Low Engagement
- [ ] Compare hook promise to content
- [ ] Check completion rate (retention)
- [ ] Review content structure
- [ ] Ensure promise fulfillment

### Growing But No Leads
- [ ] Check offer frequency
- [ ] Review offer messaging
- [ ] Analyze offer click-through
- [ ] Test lead magnet vs main offer

### Shrinking After Monetization
- [ ] Calculate actual give:ask ratio
- [ ] Compare to pre-monetization growth
- [ ] Reduce ask frequency
- [ ] Return to "give until they ask"

### Content Creation Burnout
- [ ] Use all five topic categories
- [ ] Implement repurposing
- [ ] Consider team/delegation
- [ ] Batch content creation

## Content Templates

### List Post Template
```
[Number] [Category] That [Outcome]

[Hook: Why this matters to you]

1. [Item 1]
   [Brief explanation]

2. [Item 2]
   [Brief explanation]

[Continue for all items]

[Reward: Summary + Bonus insight]
```

### Steps Post Template
```
How I [Achieve Result] in [Number] Steps

[Hook: Why you should care]

Step 1: [Action]
[Explanation + Why]

Step 2: [Action]
[Explanation + Why]

[Continue for all steps]

[Reward: What happens when you complete all steps]
```

### Story Post Template
```
[Intriguing opening line]

[Setup: Context and characters]

[Conflict: What went wrong/challenge faced]

[Journey: What happened next]

[Resolution: How it turned out]

[Lesson: What I learned that helps you]
```

### Integrated Offer Template (Long-form)
```
[After valuable section]

Quick note: If you want more on [topic], I put together
[lead magnet description] that shows [specific value].
Grab it at [destination]. Back to what we were talking about...
```

### Intermittent Offer Template (Short-form)
```
[After 10 give posts]

I'm looking for [number] [avatar] who want to [achieve result]
in [timeframe] without [sacrifice].

If that's you, [CTA].
```
