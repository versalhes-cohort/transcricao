## Overview

This checklist covers all phases of executing a high-converting online workshop launch. Target conversion: 10-30% of live attendees. Target ROI: 5-20x ad spend.

**Workshop Duration:** 3-4 hours
**Preparation Time:** 14-30 days minimum

---

## PHASE A: PRE-WORKSHOP PREPARATION

### A1. Offer & Pricing (D-30 to D-28)

#### Core Offer Definition
- [ ] Product/service clearly defined with specific deliverables
- [ ] Outcome promise is specific and measurable
- [ ] Delivery mechanism documented
- [ ] Minimum 10 existing customers with results

#### Pricing Structure
- [ ] Anchor price established (10-20x final price)
- [ ] Reduction sequence planned (minimum 3 steps)
- [ ] Final price justified by value comparison
- [ ] Payment options defined:
  - [ ] A vista com desconto
  - [ ] Parcelado sem juros
  - [ ] Entrada + parcelas
  - [ ] Gateway backup configured

#### Value Stack (Minimum 5 Components)
- [ ] Component 1: _________________ Value: $________
- [ ] Component 2: _________________ Value: $________
- [ ] Component 3: _________________ Value: $________
- [ ] Component 4: _________________ Value: $________
- [ ] Component 5: _________________ Value: $________
- [ ] **Total Stack Value:** $________ (must be 10x+ final price)

### A2. Bonus Stack (D-28 to D-25)

#### Standard Bonuses (4-6 Required)
- [ ] **Bonus 1:** Sistema/template pratico (valor 3-5x preco)
- [ ] **Bonus 2:** Treinamento complementar
- [ ] **Bonus 3:** Comunidade ou suporte
- [ ] **Bonus 4:** Ferramentas/software
- [ ] **Bonus 5:** Sessoes ao vivo/consultoria
- [ ] **Bonus 6:** Conteudo exclusivo futuro

#### Scarcity Bonuses
- [ ] Bonus apenas para os primeiros X compradores
- [ ] Bonus apenas durante o ao vivo
- [ ] Item que desaparece quando live termina (ex: Launch Blackbook)

#### Downsell Prepared
- [ ] Versao simplificada da oferta
- [ ] Parcelamento alternativo
- [ ] Trial ou teste

### A4. Social Proof Collection (D-28 to D-21)

#### Personal Results
- [ ] Faturamento comprovavel do seu negocio
- [ ] Numero de empresas vendidas (se aplicavel)
- [ ] Evolucao temporal (de X para Y em Z tempo)
- [ ] Screenshots de dashboards/relatorios
- [ ] Prints de transferencias bancarias significativas

#### Client Results (Minimum 10 Cases)
- [ ] Screenshots de depoimentos em texto
- [ ] Videos de 30-60 segundos de clientes
- [ ] Prints de conversas no WhatsApp/email
- [ ] Antes/depois com numeros especificos
- [ ] Diversidade de nichos/industrias

#### Client Story Types (Need All 5)
- [ ] Cliente que estava quebrado
- [ ] Cliente que era cetico
- [ ] Cliente que ja tinha tentado tudo
- [ ] Cliente em nicho diferente
- [ ] Cliente com resultado rapido

#### External Validations
- [ ] Certificacoes relevantes
- [ ] Aparicoes em midia
- [ ] Parcerias com empresas conhecidas
- [ ] Numero de seguidores/inscritos
- [ ] Reviews em plataformas publicas
- [ ] Premiacoes ou reconhecimentos

#### Real-Time Proof Systems
- [ ] Dashboard ao vivo de vendas configured
- [ ] Contador de participantes ao vivo ready
- [ ] System to display testemunhos durante evento
- [ ] Demonstracoes praticas tested

#### Case Study Structure (for each case)
```
- [ ] 1. Situacao inicial (numeros)
- [ ] 2. Tentativas anteriores
- [ ] 3. Ceticismo/objecoes
- [ ] 4. Implementacao especifica
- [ ] 5. Resultado mensuravel
- [ ] 6. Tempo para resultado
- [ ] 7. Efeito colateral positivo
```

### A5. Slide Production (D-20 to D-11)

#### Quantity Requirements
- [ ] Minimum 100 slides prepared
- [ ] Target: 1600 slides for full production
- [ ] Pacing: 6-10 slides per minute

#### Slide Distribution by Block
- [ ] Bloco 1 - Abertura e Gancho: Slides 1-150 (15-20 min)
- [ ] Bloco 2 - Problema e Agitacao: Slides 151-350 (20-25 min)
- [ ] Bloco 3 - Solucao e Framework: Slides 351-800 (45-60 min)
- [ ] Bloco 4 - Implementacao: Slides 801-1000 (20-30 min)
- [ ] Bloco 5 - Oferta e Stack: Slides 1001-1400 (30-40 min)
- [ ] Bloco 6 - Fechamento e Acao: Slides 1401-1600 (15-20 min)

#### Slide Type Distribution
- [ ] 40% Slides de Texto (max 7 words, 72pt+ font)
- [ ] 20% Slides de Dados (graficos simplificados)
- [ ] 20% Slides de Imagem (screenshots reais)
- [ ] 10% Slides de Transicao (titulos, perguntas)
- [ ] 10% Slides Mistos (texto sobre imagem)

#### Design Specifications
- [ ] Aspect ratio: 16:9 (1920x1080 minimum)
- [ ] Font: Sans-serif bold (Impact/Bebas/Montserrat)
- [ ] Minimum font size: 32pt
- [ ] Maximum 3 colors used
- [ ] All images high resolution, no watermarks

### A6. Scripts Preparation (D-20 to D-14)

#### Required Scripts
- [ ] Script de Abertura (primeiros 60 segundos)
- [ ] Transicao Bloco 1->2
- [ ] Transicao Bloco 2->3
- [ ] Transicao Bloco 3->4
- [ ] Transicao Bloco 4->5 (Educacao -> Oferta)
- [ ] Transicao Bloco 5->6
- [ ] Script de Reducao de Preco
- [ ] Script de Garantia
- [ ] Script de Fechamento/Urgencia

#### Key Scripts Documented

**Script de Abertura:**
```
"Nos proximos [TEMPO], vou mostrar exatamente como [RESULTADO]
sem [OBJECAO 1], sem [OBJECAO 2], mesmo que [LIMITACAO].
E vou provar isso ao vivo, fazendo [DEMONSTRACAO].
Se ao final voce nao tiver [RESULTADO GARANTIDO], eu [PENALIDADE]."
```
- [ ] Script customized with specifics

**Transicao Educacao -> Oferta:**
```
"Voce viu como [INSIGHT PRINCIPAL].
Agora voce tem duas opcoes:
Opcao 1: Tentar implementar sozinho.
Opcao 2: Deixar eu instalar isso para voce, junto com [BENEFICIO EXTRA].
Deixe-me mostrar exatamente o que voce recebe..."
```
- [ ] Script customized with specifics

**Script de Reducao de Preco:**
```
"O valor normal seria [ANCORA] porque [JUSTIFICATIVA].
Mas nao vou cobrar [ANCORA].
Nem mesmo [50%].
Nem [25%].
Hoje, para quem esta ao vivo, o investimento e [PRECO FINAL].
Mas tem um detalhe..."
[REVELAR SURPRESA]
```
- [ ] Script customized with specifics

## CONTINGENCY CHECKLISTS

### If Streaming Fails
- [ ] Switch to Zoom backup immediately
- [ ] Send email blast with new link
- [ ] Chat team directs to backup
- [ ] Continue from where you stopped
- [ ] Acknowledge issue briefly, move on

### If Payment Gateway Fails
- [ ] Switch to backup gateway
- [ ] Team begins manual payment collection
- [ ] Phone orders taken
- [ ] Collect contact info for follow-up

### If Retention Drops Below 25%
- [ ] Accelerate to value/demonstration
- [ ] Add more visual variety
- [ ] Increase interaction (polls, questions)
- [ ] Cut content that isn't landing

### If Conversion Below 5%
- [ ] Address objections more directly
- [ ] Add more case studies
- [ ] Strengthen urgency elements
- [ ] Extend Q&A time

## CRITICAL ELEMENTS CHECKLIST (Hormozi Specifics)

From the source materials, these elements are critical for success:

- [ ] **Doacao/Causa:** Element altruista that justifies action
- [ ] **Recordes/Marcos:** Something memorable and shareable
- [ ] **Plot Twist:** Unexpected element in offer (ex: free, special condition)
- [ ] **Demonstracao ao Vivo:** Real-time proof of competence
- [ ] **Multiplas Cameras:** High production value (if applicable)
- [ ] **Equipe Visivel:** Shows structure and credibility
- [ ] **Tempo Estendido:** 3+ hours shows commitment
