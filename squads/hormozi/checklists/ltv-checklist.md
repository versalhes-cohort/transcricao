## Pre-Flight Check

Before implementing the LTV framework, verify:

### Business Validation

- [ ] **Product validated** - At least 10 satisfied paying customers
- [ ] **CAC documented** - Cost of Acquisition known and consistent
- [ ] **Delivery capacity** - Can deliver 2x current volume
- [ ] **Payment system ready** - Processor accepts plans and recurring billing
- [ ] **Cash runway** - 3 months of expenses OR credit line available
- [ ] **Tracking installed** - Conversion metrics by stage

### Red Flags - NOT Ready If:

- [ ] **Refund rate > 5%** - Product issues must be fixed first
- [ ] **Can't explain value in 30 seconds** - Messaging needs work
- [ ] **No testimonials/case studies** - Social proof missing
- [ ] **Gross margin < 30%** - Unit economics don't work
- [ ] **Churn > 10% monthly** - Retention crisis

### Scope Validation

**Applies to:**
- [ ] Businesses with recurring products/services
- [ ] Medium to high ticket offers ($500+)
- [ ] Documented CAC
- [ ] B2B and B2C with defined sales cycle

**Does NOT apply to:**
- [ ] Commodities without differentiation
- [ ] Businesses without scalable delivery capacity
- [ ] Gross margin < 30%
- [ ] Refund rate > 5%

## SECTION A: Four Pillars of LTV Validation

### A.1: Pillar 1 - 30-Day Profit

- [ ] **Formula understood** - L30 = (P1 x C1) + Sum(Un x Cn x Vn) - CAC - COGS
- [ ] **CAC calculated** - Cost of Acquisition documented
- [ ] **COGS calculated** - Cost of Goods/Services documented
- [ ] **Upsell revenue projected** - Each upsell priced and conversion estimated
- [ ] **30-day profit positive** - Covers CAC + COGS in 30 days
- [ ] **Credit card strategy enabled** - Can use CC as infinite working capital
- [ ] **Consequence understood** - Negative = run out of cash, stop advertising, die slowly

**Formula Calculator:**
```

Upsell Revenue Sum: ____________
CAC: ____________
COGS: ____________
30-Day Profit (L30): ____________
[ ] PASS (L30 > 0)
[ ] FAIL (fix offer or add upsells)
```

### A.2: Pillar 2 - Problem-Solution Sequence

- [ ] **Core problem identified** - What your main offer solves
- [ ] **Revealed problems mapped** - Each solution creates new problems
- [ ] **Upsell chain designed** - Each problem has a solution product
- [ ] **Chain is 3+ deep** - Multiple upsell opportunities exist
- [ ] **Problems are genuine** - Not manufactured or forced
- [ ] **Solutions provide real value** - Actually help customers
- [ ] **Consequence understood** - Ignoring = lose 80% of potential profit

**Problem-Solution Examples Validated:**
```
[ ] Rented car -> needs insurance (problem created)
[ ] Fitness challenge -> needs supplements (problem created)
[ ] Software -> needs training (problem created)
```

### A.3: Pillar 3 - Value Anchoring

- [ ] **Anchor price defined** - Premium option 5-10x main price
- [ ] **Premium is genuine** - Would happily fulfill if purchased
- [ ] **Presentation order correct** - Most expensive shown first
- [ ] **Psychological recalibration works** - Other options seem cheap
- [ ] **Consequence understood** - Ignoring = customers always buy cheapest option

**Suit Store Case Study Validated:**
```
[ ] Customer budget: $500
[ ] First suit presented: $16,000
[ ] Second suit presented: $2,200
[ ] Customer bought: $2,200 + $300 accessories = $2,500 (5x budget)
```

### A.4: Pillar 4 - Zero Friction

- [ ] **Card on file enabled** - Can charge without re-asking
- [ ] **No "do you want to buy?"** - Ask "which card?" instead
- [ ] **Payment barriers removed** - Nothing between decision and charge
- [ ] **Multiple payment options** - Credit, debit, financing available
- [ ] **Consequence understood** - 50% of closed sales die in payment friction

- [ ] Value = 3-5x delivery cost

**Timeline Selection:**
- [ ] Short (21-28 days) - Maximum urgency
- [ ] Medium (6-8 weeks) - Ideal balance
- [ ] Long (12-16 weeks) - Deep transformations

**Credit Application:**
- [ ] Option A - 100% applied to next purchase immediately

**Formula Applied:**
- [ ] "You can't have X without Y" framework documented

**9 Key Tactics:**
- [ ] Most profitable upsells first

- [ ] Surprise and delight - Give ALL bonuses even if early yes
- [ ] Hyper-buyer detection - Weddings, babies, new hobbies = embrace upsells
- [ ] Free bonuses CREATE problems that upsells solve
- [ ] Faster access = higher value
- [ ] Bundle upsells and NAME them (transformation package)

- [ ] Average 2-4x LTV increase when well positioned

### D.2: Offer #7 - Menu Upsell

**Formula:** Unsell -> Prescribe -> A/B -> Card on File

**Four Tactics Present:**
- [ ] **UNSELLING** - Cross out what they DON'T need -> builds goodwill -> 25% to 75% conversion
- [ ] **PRESCRIPTION** - Tell what they need, how to use, when, how much
- [ ] **A/B CHOICE** - "Chocolate or vanilla?" removes "yes/no" -> both choices = sale
- [ ] **CARD ON FILE** - "Use the card on file?" -> 40% conversion increase

**Economist Play Ready:**
- [ ] A ($59), B ($125), A+B ($125) -> 80% choose combo

**Real Numbers Validated:**
- [ ] $0 to $5-10k/day supplement sales using menu upsell
- [ ] 80%+ conversion when done correctly
- [ ] Unselling: 25% -> 75% conversion (3x)

### D.3: Offer #8 - Anchor Upsell

**Formula:** Premium first -> Gasp -> Rescue -> Main offer -> Payment

**5 Steps:**
- [ ] Step 1: Present the anchor (5-10x price)
- [ ] Step 2: Wait for the gasp
- [ ] Step 3: Come to rescue: "Voce se importa muito com [premium feature]?"
- [ ] Step 4: Present main offer
- [ ] Step 5: Ask how they want to pay

**CRITICAL:**
- [ ] Treat anchor as REAL offer - If fake, lose trust
- [ ] Quote understood: "The only thing worse than making a $1,000 offer to someone with a $100 budget is making a $100 offer to someone with a $1,000 budget"

**Base Calculation Validated:**
```
Without anchor: 30 buy at $2,200 = $66,000
With anchor: 15 premium at $16k + 45 main at $2.2k = $240k + $99k = $339k
Lift: 5.1x revenue
[ ] VALIDATED
```

## SECTION E: Stage III - Downsell Offers (Turn No's into Yes's) Validation

### E.1: Downselling Rules

- [ ] **Rule 1** - "No" means no to THIS offer, not all offers
- [ ] **Rule 2** - Downsells are TRADES - give something, get something
- [ ] **Rule 3** - Personalize, don't pressure
- [ ] **Rule 4** - Same thing in NEW ways
- [ ] **Rule 5** - NEVER lower price on same thing - change offer with price
- [ ] **Rule 6** - Customers talk about price - consistency matters

### E.2: Offer #10 - Payment Plan Downsell

**7-Step Process (stop when they buy):**
- [ ] Step 1: Reward paying in full (price WITH interest -> prepay = discount)
- [ ] Step 2: Third-party financing / credit card / layaway
  - [ ] "Do you prefer that I decide your terms or YOU decide?" -> "I decide" -> card
- [ ] Step 3: Half now, half later (align with pay day)
- [ ] Step 4: CHECK: Scale 1-10. 8+ continue. 7- switch to feature downsell
- [ ] Step 5: Divide into three payments
- [ ] Step 6: Spread evenly (e.g., 16 weeks = 16 payments)
- [ ] Step 7: Free trial (-> Trial with Penalty)

**Seesaw Technique Script:**
- [ ] "Giant or small monthly payments?" -> "Small"
- [ ] "Prepay today: massive discount + zero monthly. Does that work?"
- [ ] If no: "The more you give now = smaller monthly payments"

**Churn by Billing (14,000 companies via ProfitWell):**
```
Monthly:     10.7% churn/month
Quarterly:   5% churn/month
Annual:      2% churn/month
RULE: Double period = half churn. Always start high.
[ ] VALIDATED
```

**Base Calculation Validated:**
```
Without plan: 30 buy at $1,000 = $30,000
With plan downsell: 30 full + 20 plans (80% complete) = $30k + $16k = $46,000
Lift: 53% revenue increase
[ ] VALIDATED
```

### E.3: Offer #11 - Trial with Penalty

**Formula:** "If you do X, Y, Z, I'll let you start for free."

**5 Steps:**
- [ ] Step 1: Offer trial LAST (only after main rejected)
- [ ] Step 2: ALWAYS get card ("Which card?" / "How we've always done it")
- [ ] Step 3: Sell staying and paying ("If it works, will you stay long-term?")
- [ ] Step 4: Explain fees AFTER card (not before -> more resistance)
- [ ] Step 5: MANDATORY check-ins (your upsell opportunities)

**Post-Trial Handling:**
- [ ] **Loved it:** Easy upsell to bigger/longer service
- [ ] **Hated it:** "You're right, I'm upset. Your needs match our premium." (50% buy)
- [ ] **Didn't use:** Contact multiple times, waive fee if they show up

**Base Calculation Validated:**
```
100 leads: 30 buy main $1,000 + 40 trial
8 fail criteria ($200 penalty) + 24 convert ($500/month)
30 days: $30,000 + $1,600 + $12,000 = $43,600 -> 45% lift
[ ] VALIDATED
```

### E.4: Offer #12 - Feature Downsell

**Formula:** Remove feature -> lower price -> "How about now?"

**The Guarantee Hack:**
```
Full offer $1,000 with guarantee: 25 buy
Remove guarantee, $600: some buy THIS
BUT many reconsider -> buy FULL $1,000 because NOW they see guarantee value
Result: 35 full + 40 without = 75 total (was 25) -> 3x close rate
[ ] VALIDATED
```

**What to Remove (highest value first to trigger re-upsell):**
- [ ] Quantity
- [ ] Quality
- [ ] Service level
- [ ] Availability
- [ ] Speed
- [ ] Ratio
- [ ] Communication
- [ ] Provider
- [ ] Live vs recorded
- [ ] DIY vs DFY
- [ ] Personalization
- [ ] Insurance/Guarantee
- [ ] Entire features

**Base Calculation Validated:**
```
Before: 25 x $1,000 = $25,000
After: 35 x $1,000 + 40 x $600 = $59,000 -> 2.36x lift
40% of downsell buyers eventually upgrade
[ ] VALIDATED
```
