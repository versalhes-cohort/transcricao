## Quick Reference: The Core Four Matrix

| | Warm (Know You) | Cold (Strangers) |
|---|-----------------|------------------|
| **1-to-1 (Private)** | #1 Warm Outreach | #3 Cold Outreach |
| **1-to-many (Public)** | #2 Post Content | #4 Paid Ads |

**Core Equation:** More Advertising → More Engaged Leads → More Money

---

## Pre-Flight Checklist

### Before Starting Any Core Four Method

- [ ] **Offer/Lead Magnet Ready**
  - [ ] Solves a specific problem for a specific audience
  - [ ] Clear dream outcome defined
  - [ ] Value equation elements documented
  - [ ] Could charge for it (even if giving free)

- [ ] **Tracking System Ready**
  - [ ] CRM or spreadsheet for contacts
  - [ ] Way to track outreach attempts
  - [ ] Way to track responses
  - [ ] Way to track conversions

- [ ] **Resource Assessment Complete**
  - [ ] Hours available per day documented
  - [ ] Budget available documented
  - [ ] More time than money? → Start Warm Outreach / Content
  - [ ] More money than time? → Start Cold Outreach / Paid Ads

## #1 Warm Outreach Checklist

### Setup Phase

- [ ] **List Building Complete**
  - [ ] Phone contacts exported
  - [ ] Email contacts exported (ALL accounts)
  - [ ] Facebook friends counted
  - [ ] Instagram followers counted
  - [ ] LinkedIn connections counted
  - [ ] Twitter/X followers counted
  - [ ] Other platforms checked
  - [ ] Total contacts documented: _______

- [ ] **Platform Priority Set**
  - [ ] Platforms ranked by contact count
  - [ ] First platform selected: ____________
  - [ ] Plan to hit all platforms eventually

### Daily Execution (Rule of 100)

- [ ] **Day ___ of 100**
  - [ ] 100 personalized messages sent
  - [ ] Each message includes personal detail
  - [ ] Social profiles checked for recent events
  - [ ] Follow-ups sent (Day 2 & 3 for non-responders)

### Response Handling

- [ ] **A-C-A Framework Applied**
  - [ ] Acknowledged what they said
  - [ ] Complimented (tied to character trait)
  - [ ] Asked question toward offer topic
  - [ ] 3-4 exchanges before making offer

- [ ] **Offer Made Correctly**
  - [ ] Asked "Do you know anyone who..." (not "Do you need...")
  - [ ] Described struggles clearly
  - [ ] Stated dream outcome
  - [ ] Stated time delay
  - [ ] Stated effort/sacrifice avoided
  - [ ] Included social proof (examples)
  - [ ] Used "backup humor" if they said no

### Free Client Phase (First 5)

- [ ] **Free Offer Structure**
  - [ ] Limited to 5 clients
  - [ ] Commitment 1: Use it
  - [ ] Commitment 2: Give feedback
  - [ ] Commitment 3: Leave review if deserved
  - [ ] "Does that sound fair?" asked

- [ ] **Client Tracking**
  - [ ] Free client #1: ____________
  - [ ] Free client #2: ____________
  - [ ] Free client #3: ____________
  - [ ] Free client #4: ____________
  - [ ] Free client #5: ____________

### Transition to Paid

- [ ] **Referrals Started** (trigger to charge)
- [ ] **Price Escalation**
  - [ ] Next 5: 80% off
  - [ ] Next 5: 60% off
  - [ ] Next 5: 40% off
  - [ ] Next 5: 20% off
  - [ ] Full price
  - [ ] Continue increasing 20% every 5

### List Maintenance

- [ ] **Regular Value Delivery**
  - [ ] Email/social content schedule set
  - [ ] 9-word emails sent periodically
  - [ ] Hot leads prioritized from responses

## #2 Post Free Content Checklist

### Setup Phase

- [ ] **Platform Strategy Selected**
  - [ ] Depth-first (maximize one, then add)
  - [ ] Breadth-first (add all, then maximize)
  - [ ] First platform selected: ____________

- [ ] **Content Schedule Set**
  - [ ] Minimum: Daily posting
  - [ ] Time commitment: 100 minutes/day
  - [ ] Duration: 100 days minimum

### Content Creation (Unit of Content)

- [ ] **Hook Created**
  - [ ] Topic selected (Past/Present/Trends/Manufactured)
  - [ ] Headline includes 2+ elements:
    - [ ] Recency
    - [ ] Relevance
    - [ ] Celebrity
    - [ ] Proximity
    - [ ] Conflict
    - [ ] Unusual
    - [ ] Ongoing
  - [ ] Format matches platform norms

- [ ] **Retention Built**
  - [ ] Structure selected:
    - [ ] Lists (items with theme)
    - [ ] Steps (actions in order)
    - [ ] Stories (events with lessons)
  - [ ] Curiosity embedded (unanswered questions)

- [ ] **Reward Delivered**
  - [ ] Promise from hook fully satisfied
  - [ ] Audience would want to share
  - [ ] Audience would want more

### Give:Ask Ratio

- [ ] **Ratio Tracked**
  - [ ] Current ratio: ___ give : ___ ask
  - [ ] Minimum target: 3:1
  - [ ] Ideal: Give until they ask

- [ ] **CTA Strategy Selected** (if asking)
  - [ ] Integrated (ads within content)
  - [ ] Intermittent (mostly give, occasional ask)
  - [ ] CTA type: Lead magnet / Direct offer

### Measurement

- [ ] **Weekly Tracking**
  - [ ] Posts published: ___
  - [ ] Engagement rate: ___%
  - [ ] Leads generated: ___

- [ ] **Monthly Tracking**
  - [ ] Audience size start: ___
  - [ ] Audience size end: ___
  - [ ] Growth: ___% MoM
  - [ ] Growth indicates content quality

## #3 Cold Outreach Checklist

### Setup Phase

- [ ] **Target List Built**
  - [ ] Ideal customer profile defined
  - [ ] List source selected:
    - [ ] Purchased lists
    - [ ] Scraped data
    - [ ] Industry databases
    - [ ] Conference lists
    - [ ] Association lists
    - [ ] Social media searches
  - [ ] List quality verified
  - [ ] Total contacts: _______

- [ ] **Automation Setup**
  - [ ] Email automation tool ready
  - [ ] SMS automation tool ready (if applicable)
  - [ ] DM automation tool ready (if applicable)
  - [ ] Compliance requirements reviewed

### Message Development

- [ ] **Cold Message Template**
  - [ ] Relevance: Why contacting THEM specifically
  - [ ] Value: What's in it for them
  - [ ] Credibility: Why believe you
  - [ ] CTA: Clear, low-friction next step
  - [ ] Multiple variations created for testing

### Daily Execution (Rule of 100)

- [ ] **Day ___ of 100**
  - [ ] 100+ cold outreaches sent (automated)
  - [ ] Channels used:
    - [ ] Email
    - [ ] Text
    - [ ] DM
    - [ ] Cold call
    - [ ] Flyers
  - [ ] Platform rules followed

### Response Handling

- [ ] **Response Received**
  - [ ] Treated like gold
  - [ ] A-C-A framework applied
  - [ ] Moved to warm outreach process
  - [ ] Tracked in CRM

### Measurement

- [ ] **Weekly Tracking**
  - [ ] Outreaches sent: ___
  - [ ] Responses received: ___
  - [ ] Response rate: ___% (target >5%)
  - [ ] Conversions: ___

## #4 Paid Ads Checklist

### Phase 1: Track Money

- [ ] **Tracking Setup Complete**
  - [ ] Platform tracking installed
  - [ ] Conversion events defined
  - [ ] Attribution configured
  - [ ] Dashboard/reporting ready
  - [ ] **DO NOT SPEND UNTIL TRACKING WORKS**

### Phase 2: Lose Money (Test)

- [ ] **Ad Creation**
  - [ ] Ad variations created (minimum 10)
  - [ ] Hook-Retain-Reward structure applied
  - [ ] Best free content considered for ads
  - [ ] Multiple creatives tested

- [ ] **Budget Set**
  - [ ] 30-day customer profit calculated: $___
  - [ ] Test budget per ad: $_____ (2x 30-day profit)
  - [ ] Cutoff if zero leads: $_____ (1x 30-day profit)

- [ ] **Testing**
  - [ ] Ads launched
  - [ ] Performance monitored daily
  - [ ] Losers cut at cutoff
  - [ ] Winners identified

### Phase 3: Print Money (Scale)

- [ ] **Winner Validation**
  - [ ] LTGP calculated: $_____
  - [ ] CAC calculated: $_____
  - [ ] LTGP:CAC ratio: ___:1 (target >3:1)

- [ ] **Scaling Budget**
  - [ ] Desired customers: ___
  - [ ] Base budget: Customers × CAC = $_____
  - [ ] Add 20% buffer: $_____
  - [ ] Daily budget: $_____

- [ ] **Customer-Financed Acquisition**
  - [ ] 30-day revenue per customer: $_____
  - [ ] 30-day delivery cost: $_____
  - [ ] 30-day profit: $_____
  - [ ] CAC: $_____
  - [ ] 30-day profit >= CAC? (Yes = unlimited scale)
  - [ ] If no: Upsells/cross-sells added to achieve

### Diagnostic (If Not Profitable)

- [ ] **CAC vs Industry Average**
  - [ ] Industry average CAC: $_____
  - [ ] Your CAC: $_____
  - [ ] If CAC > 3x industry: Fix ads
  - [ ] If CAC < 3x industry: Fix model (LTGP too low)

## More-Better-New Optimization Checklist

### When to Apply
- [ ] Current method has plateaued
- [ ] Returns are diminishing
- [ ] Want to grow faster

### MORE Phase

- [ ] **Volume Check**
  - [ ] Current daily actions: ___
  - [ ] Rule of 100 target: 100
  - [ ] Gap: ___
  - [ ] Action: Increase to 100 before anything else

### BETTER Phase

- [ ] **Constraint Identification**
  - [ ] Funnel steps mapped
  - [ ] Drop-off rates calculated:
    - [ ] Step 1: ___% pass
    - [ ] Step 2: ___% pass
    - [ ] Step 3: ___% pass
  - [ ] Biggest drop-off = constraint: _______

- [ ] **Weekly Testing**
  - [ ] Week of: _______
  - [ ] Test hypothesis: _______
  - [ ] Single variable changed: _______
  - [ ] Results: Win / Lose
  - [ ] Logged in test register: Yes / No

- [ ] **Move to Next Constraint**
  - [ ] Current constraint tested 4 times
  - [ ] Unable to beat current version
  - [ ] Moving to next biggest drop-off

### NEW Phase (Only after MORE+BETTER exhausted)

- [ ] **Priority Order**
  1. [ ] New placements on same platform
  2. [ ] Same approach on new platform
  3. [ ] New Core Four method entirely

- [ ] **Decision Criteria**
  - [ ] Returns from MORE+BETTER < Returns from NEW
  - [ ] Resources available for new channel
  - [ ] Selected: _______

## Scripts Quick Reference

### Warm Outreach - The Ask
```
"By the way, do you know anyone who is [struggles]
looking for [dream outcome] in [time delay]?

I'm taking on five case studies for free...
I help them [dream outcome] without [effort/sacrifice].
I even guarantee [dream outcome] or work until they do.

Does anyone you like come to mind?
...Haha, does anyone you hate come to mind?"
```

### Free Offer Acceptance
```
"I'll give everything for free, as long as you promise:
1) Use it
2) Give me feedback
3) Leave a killer review if deserved

Does that sound fair?"
```

### 9-Word Re-engagement
```
"Are you still looking for [4-word desire]?"
```

### Content CTA - Offer
```
"I'm looking for 5 [avatar] to help achieve [outcome] in [time].
You don't need to [effort/sacrifice].
If you don't get [outcome], I'll refund AND work until you do.
DM me / Comment / Reply if interested."
```

## Emergency Quick Actions

### "I have zero leads"
1. Start Warm Outreach TODAY
2. 100 reach outs per day
3. Free offer for first 5
4. Continue for 100 days

### "My paid ads are losing money"
1. Check tracking (is it accurate?)
2. Calculate LTGP:CAC
3. If CAC > 3x industry: Fix ads
4. If CAC normal: Fix model (add upsells)
5. Implement customer-financed acquisition

### "My content isn't growing audience"
1. Review hooks (2+ elements?)
2. Check if reward matches promise
3. Compare format to top performers
4. Increase posting frequency
5. Consider depth-first before breadth

### "I hit a plateau"
1. Are you at 100/day? (MORE)
2. Are you testing weekly? (BETTER)
3. Have you tried new placements? (NEW)
4. Follow priority: MORE → BETTER → NEW
