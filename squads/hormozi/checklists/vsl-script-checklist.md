## Phase 1: Hook (First 15-30 Seconds)

### Pattern Interrupt Opening

| Checkpoint | Status | Quality Notes |
|------------|--------|---------------|
| [ ] Addresses viewer directly (you/your) | | |
| [ ] Creates immediate curiosity or shock | | |
| [ ] Matches VSL type template | | |
| [ ] Under 5 seconds to hook | | |
| [ ] No wasted words/filler | | |

### Curiosity Gap

| Checkpoint | Status | Quality Notes |
|------------|--------|---------------|
| [ ] Teases value without revealing | | |
| [ ] Creates anticipation for what's coming | | |
| [ ] Gap will be paid off later in video | | |
| [ ] Not clickbait (delivers on promise) | | |

### Hook Templates by VSL Type

**Registration VSL:**
```
[ ] Uses "quick question" or direct ask pattern
[ ] Invites to specific event/opportunity
[ ] Mentions scale/credibility quickly
```

**VIP Upgrade VSL:**
```
[ ] Confirms registration/purchase first
[ ] Transitions to "bonus" or "upgrade" offer
[ ] Numbers the bonuses (1, 2, 3, 4)
```

**Recap VSL:**
```
[ ] Creates urgency immediately ("disappears forever")
[ ] Acknowledges limited time
[ ] Separates audience (beginners vs business owners)
```

**OTO VSL:**
```
[ ] Thanks for previous action
[ ] Confirms what's coming to them
[ ] Transitions to "special offer" they were promised
```

**Thank You VSL:**
```
[ ] Validates purchase decision
[ ] Creates excitement about what they got
[ ] Sets up next steps
```

## Hormozi VSL Golden Rules

### Rule 1: Control the Timing
```
"VSL controls o tempo de exposicao a mensagem. A person who watches
a VSL cannot skim. They experience your pitch at YOUR pace."
```

### Rule 2: One-on-One Conversation
```
"A VSL is not a video commercial. It's a 1-on-1 sales conversation
filmed once and delivered infinitely."
```

### Rule 3: Natural Delivery
```
"If you can't say it naturally to one person across a table,
don't put it in a VSL."
```

### Rule 4: Real Urgency Only
```
"Real urgency amplifies. Fake urgency destroys."
```

### Rule 5: Stack the Proof
```
"Proof depth must match price point. The higher the ask,
the more evidence required."
```
