## Pre-Naming Checklist

### Preconditions Verified
- [ ] Offer deliverable and outcome clearly defined
- [ ] Ideal Customer Profile (ICP) documented
- [ ] Price point established
- [ ] Specific result promise quantified
- [ ] Timeframe for results known
- [ ] Unique mechanism identified (the "how")

---

## MAGICO Framework Checklist

### Components Present (2+ Required)
- [ ] **M** - Mecanismo unico (unique mechanism/method)
- [ ] **A** - Alvo atraente (attractive target/outcome)
- [ ] **G** - Grafia diferente (creative spelling/format)
- [ ] **I** - Ideia que ajuda (solution concept)
- [ ] **C** - Convergencia temporal (time element)
- [ ] **O** - Objetivo final (dream result)

### MAGICO Quality Check
- [ ] Contains 2+ MAGICO components
- [ ] Components work together cohesively
- [ ] Name tells prospect what they get in <10 words

## Archetype Selection Checklist

### Context Analysis
- [ ] ICP technical level assessed
- [ ] Category maturity identified
- [ ] Primary barrier to purchase known
- [ ] Competitive landscape mapped

### Archetype Match (Select One)
- [ ] **LITERAL EXTREMO** - Non-technical audience, complex category
- [ ] **VERBO CULTURAL** - Creating new behavior, want dominance
- [ ] **ESTADO DESEJADO** - Emotional transformation product
- [ ] **NUMERICO ESPECIFICO** - Time/quantity as differentiator
- [ ] **PARADOXO INTRIGANTE** - Commodity category, need attention
- [ ] **TRIBAL IDENTIFIER** - Community > product
- [ ] **METAFORA VISUAL** - Visual/physical benefit
- [ ] **STATUS SIGNAL** - Premium pricing, status buyers
- [ ] **INVENTOR FRIENDLY** - New category, no reference
- [ ] **HERITAGE FABRICADO** - Trust is main barrier

## 7 Immutable Laws Check

### Laws Compliance
- [ ] **Law 1**: Simplicidade > Criatividade (Simple beats clever)
- [ ] **Law 2**: Beneficio > Feature (Benefit beats feature)
- [ ] **Law 3**: Emocao > Logica (Emotion beats logic)
- [ ] **Law 4**: Especifico > Generico (Specific beats generic)
- [ ] **Law 5**: Curto > Longo (Short beats long - avg 7.3 chars)
- [ ] **Law 6**: Historia > Descricao (Story beats description)
- [ ] **Law 7**: Timing > Perfeicao (80% launched > 100% never)

## Specificity Checklist

### Quantified Elements
- [ ] Specific number for outcome (not range)
- [ ] Broken number used if possible (19, 67, 93 > 20, 70, 90)
- [ ] Time element specific (17 weeks > "a few months")
- [ ] Result measurable and verifiable

## Quick Reference: The MAGICO Formula

```
[TIME ELEMENT] + [UNIQUE MECHANISM] + [DREAM RESULT]

Example: "O Sistema de 6 Semanas Six Pack FRAQUEZA-A-RIQUEZA"

- 6 Semanas = Time (C)
- Sistema = Mechanism (M)
- Six Pack = Target (A)
- FRAQUEZA-A-RIQUEZA = Creative spelling (G) + Objective (O)
```
