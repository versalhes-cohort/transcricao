## Pre-Flight Checklist

Before starting lead magnet creation, confirm:

- [ ] Core offer is defined and sellable
- [ ] Target customer is specifically identified
- [ ] List of 5+ customer problems documented
- [ ] Delivery capability exists or can be built
- [ ] Testing mechanism available (audience or ad platform)
- [ ] Budget for delivery cost calculated

---

## Step 1: Problem Discovery

### Problem-Solution Cycle Validation

- [ ] Core offer (broad solution) documented
- [ ] All narrow problems listed (before broad solution needed)
- [ ] Each narrow problem assessed for:
  - [ ] Narrowness (specific, not broad)
  - [ ] Significance (people care about solving it)
  - [ ] Revelation strength (solving it reveals core offer problem)
- [ ] Winning problem selected with clear rationale

### Quality Gate QG-001
```
Pass Criteria:
- [ ] Problem is narrow (not "help them succeed" but "show them their home value")
- [ ] Problem reveals need for core offer when solved
```

**Source:** "$100M Leads p.138-144"

## Step 2: Type Selection

### Type Assessment

- [ ] Audience awareness level assessed:
  - [ ] Do they know they have this problem?
  - [ ] Are they actively seeking solution?

- [ ] Core offer structure analyzed:
  - [ ] Is it recurring solution to recurring problem?
  - [ ] Is it multi-step process?
  - [ ] Is it one-time solution?

### Type Selection Matrix

| If... | Select... | Because... |
|-------|-----------|------------|
| Audience unaware of problem | Reveal Problems | Must create awareness first |
| Core offer = recurring solution | Samples & Trials | Let them experience, then continue |
| Core offer = multi-step | One Step of Multi-Step | Prove value on one step |

- [ ] Rationale documented

### Quality Gate QG-002
```
Pass Criteria:
- [ ] Type matches audience awareness level
- [ ] Type matches core offer structure
```

## Step 6: Quality Validation

### "Could Charge For It" Test

- [ ] Would someone pay $XX for this?
- [ ] If sold, would it have 4+ star reviews?
- [ ] Does it solve the problem COMPLETELY?

### Secrets Check

- [ ] Am I withholding valuable information?
- [ ] Have I given "the how" not just "the what"?
- [ ] Would a competitor be nervous about this being free?

### Reputation Impact Assessment

- [ ] Would non-buyers speak positively about this?
- [ ] Would this build trust with the 99% who won't buy?
- [ ] Is this the quality I want my brand known for?

### Quality Gate QG-006
```
Pass Criteria:
- [ ] Could definitely charge for it
- [ ] Secrets given, not withheld
- [ ] Would build (not destroy) reputation
```

**Philosophy:** "Give away the secrets, sell the implementation."

**Source:** "$100M Leads p.257-267"

## Final Validation: The Four Things

A good lead magnet does FOUR things:

1. [ ] **Engages ideal customers when they see it**
   - Name/title tested and winning
   - Visual appealing to target audience

2. [ ] **Gets more people to engage than core offer alone**
   - Lower barrier than paid offer
   - Valuable enough to trade attention/info for

3. [ ] **Is valuable enough that they consume it**
   - Multiple formats available
   - Easy to access and use
   - Not too long/complex

4. [ ] **Makes the right people more likely to buy**
   - Reveals problem core offer solves
   - Builds trust through value delivery
   - CTA present and compelling

**Source:** "$100M Leads p.318-324"
