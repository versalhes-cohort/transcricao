## Pre-Work Checklist

### Before Using Any Angle

- [ ] Product/offer clearly defined
- [ ] Target avatar documented with exact language
- [ ] Awareness levels mapped for campaign
- [ ] Proof elements collected (testimonials, case studies, numbers)
- [ ] Read ad-angles-sop.md completely
- [ ] Understand relationship to hooks-sop.md and goated-ads-sop.md
- [ ] Testing infrastructure ready (ad platform, tracking, spreadsheet)

### Red Flags - STOP if:

- [ ] Product still being defined
- [ ] No avatar language documentation
- [ ] No proof elements available
- [ ] Haven't read companion SOPs
- [ ] Expecting to copy angles directly without adaptation

## Angle Selection Checklist

### Step 1: Determine Awareness Level

| Level | Description | Check if Targeting |
|-------|-------------|-------------------|
| 1 - Unaware | Doesn't know they have a problem | [ ] |
| 2 - Problem-Aware | Knows problem, not solutions | [ ] |
| 3 - Solution-Aware | Knows solutions exist, not your product | [ ] |
| 4 - Product-Aware | Knows your product, not convinced | [ ] |
| 5 - Most Aware | Knows product, ready to buy | [ ] |

### Step 2: Select Angle Category

**For Level 1-2 (Cold Traffic):**
- [ ] Rumors Are True (RT-01 to RT-06)
- [ ] Question Hooks (QH-01 to QH-04)
- [ ] Comparison 95% vs 5% (CC-07)

**For Level 3 (Warm Traffic):**
- [ ] Comparison/Contrast (CC-01 to CC-08)
- [ ] Systems/Framework (SF-01 to SF-06)
- [ ] Transformation (TR-01 to TR-06)

**For Level 4 (Warmer Traffic):**
- [ ] Book/Knowledge (BK-01 to BK-05)
- [ ] Transformation with Proof (TR-03, TR-04)

**For Level 5 (Hot Traffic):**
- [ ] Launch/Event (LA-01 to LA-14)
- [ ] Trial/Freemium (TF-01 to TF-14)
- [ ] Retargeting (RT-01 to RT-10)

### Step 3: Apply 70-20-10 Rule

- [ ] 70% of angles from proven winners (your past data)
- [ ] 20% from adjacent categories (same awareness, different angle type)
- [ ] 10% completely new/experimental

## Testing Checklist

### Before Each Test

- [ ] Angle fully adapted (no brackets remaining)
- [ ] 3-5 variations ready
- [ ] Budget allocated (minimum per goated-ads-sop.md)
- [ ] Success metrics defined (CTR, CPA, ROAS targets)
- [ ] Testing log prepared

### During Test (Daily)

- [ ] Performance data collected
- [ ] CTR checked (threshold: >1%)
- [ ] CPA checked (threshold: within target)
- [ ] Frequency monitored (threshold: <5)
- [ ] Comments/engagement reviewed

### After Test (Day 7)

- [ ] Top 10% performers identified
- [ ] Bottom 10% eliminated
- [ ] Middle performers noted for potential iteration
- [ ] Patterns documented
- [ ] Testing log updated (Appendix C format)

### Scale Decision

**Ready to Scale if:**
- [ ] ROAS >3x for 7 days
- [ ] CPA stable for 100+ conversions
- [ ] Frequency <3 after 14 days
- [ ] CTR maintains >1%

**Do NOT Scale if:**
- [ ] CTR <0.5% (kill)
- [ ] CPA doubled (pause)
- [ ] Frequency >10 (refresh)
- [ ] ROAS <2x for 3 days (analyze)

## Weekly Angle Creation Checklist

### Friday Ad Creation Session (3-4 hours)

**Hour 1: Review & Plan (60 min)**
- [ ] Pull last week's performance data
- [ ] Identify top 10% and bottom 10%
- [ ] Document pattern insights
- [ ] Select 15 angles for this week:
  - [ ] 10-11 from proven winners (70%)
  - [ ] 3 from adjacent categories (20%)
  - [ ] 1-2 completely new (10%)

**Hours 2-3: Adapt (90 min)**
- [ ] Adapt selected angles using worksheet
- [ ] Create 3-5 variations per angle
- [ ] Run quality checks on each
- [ ] Total output: 45-75 adapted angles

**Hour 4: Validate & Schedule (60 min)**
- [ ] Quality gate all adaptations
- [ ] Schedule tests in ad platform
- [ ] Update testing log
- [ ] Document this week's angle selections

## Troubleshooting Checklist

### If All Angles Underperforming

- [ ] Check avatar language match
- [ ] Verify proof elements are strong
- [ ] Test different awareness level
- [ ] Review competitor angles
- [ ] Escalate to marketing lead

### If Specific Category Underperforming

- [ ] Try different angles from same category
- [ ] Try adjacent category for same awareness
- [ ] Review category-specific decision rules
- [ ] Document learnings

### If Angle Suddenly Stops Working

- [ ] Check creative fatigue (>10-14 days?)
- [ ] Check competitor activity
- [ ] Check for platform algorithm changes
- [ ] Create fresh variations
- [ ] Consider pause and restart

### If No Clear Winner After 15 Tests

- [ ] Review testing methodology
- [ ] Ensure sufficient sample size
- [ ] Check for external factors
- [ ] Escalate to strategy review
- [ ] Consider broader angle pivot
