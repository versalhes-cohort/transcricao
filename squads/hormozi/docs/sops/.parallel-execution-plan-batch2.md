**Source:** `$100M Fast Cash Playbook.md` (53 KB)

**Contexto:** Playbook para gerar caixa rápido quando precisa de dinheiro AGORA.

**Focus Areas Esperados:**
- Estratégias de monetização imediata
- Ofertas de emergência (flash sales, liquidações)
- Reativação de clientes dormentes
- Upsells e cross-sells rápidos
- Táticas de cash flow de curto prazo
- Sequência de prioridades (o que fazer primeiro)

**Expected Agents:**
- cash-coordinator (orchestrator)
- opportunity-scanner (tier 0) - identifica oportunidades imediatas
- dormant-reactivator (tier 1) - reativa clientes antigos
- flash-offer-creator (tier 1) - cria ofertas de emergência
- upsell-executor (tier 1) - executa upsells
- cash-tracker (tier 2) - monitora resultados

**Decision Rules Esperados:**
- Priorização: qual tática primeiro baseado em situação
- Thresholds: quando usar cada estratégia
- Kill switches: quando parar táticas que não funcionam
