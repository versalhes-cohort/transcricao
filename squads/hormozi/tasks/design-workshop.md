## Purpose
Design a 3-4 hour workshop that converts 10-30% of live attendees into buyers.

> "50-60% of all sales happen in the last 4 hours of any campaign. A workshop concentrates all that urgency into one event." — Alex Hormozi

## The 7-Block Framework

### Block 1: Hook Impossivel (10 min)
**Purpose:** Promise + stakes + live proof element

Elements:
- World record or impossible promise
- Live demonstration element
- Stakes established
- Curiosity gap created

Script template:
```
"Nos proximos [TEMPO], vou mostrar exatamente como [RESULTADO]
sem [OBJECAO 1], sem [OBJECAO 2], mesmo que [LIMITACAO].

E vou provar isso ao vivo, na sua frente, fazendo [DEMONSTRACAO]."
```

### Block 2: Origem e Conexao (15 min)
**Purpose:** Personal story + discovery + first transformation

Elements:
- Starting point WORSE than audience
- Moment of discovery
- First transformation
- "If I did it, you can too"

### Block 3: Educacao Profunda (60 min)
**Purpose:** Framework + 3 insights + demonstration

Elements:
- Proprietary framework (named)
- 3 counter-intuitive insights
- Practical demonstration
- Genuine value delivery

**Rule:** 60%+ of workshop time is education

### Block 4: Casos em Cascata (30 min)
**Purpose:** Personal + 3-5 clients + live case

Case study structure:
1. Initial situation (numbers)
2. Previous attempts
3. Skepticism/objections
4. Specific implementation
5. Measurable result
6. Time to result
7. Positive side effect

### Block 5: Stack de Valor (20 min)
**Purpose:** High anchor + sequential reductions + bonuses

Price reduction sequence:
```
"O valor normal disso seria [ANCORA ALTA]...
Mas nao vou cobrar [ANCORA].
Nem mesmo [50% DA ANCORA].
Nem [25% DA ANCORA].
Hoje, apenas para quem esta ao vivo: [PRECO FINAL]."
```

**Rule:** Stack total must be 10x+ asking price

### Block 6: Inversao de Risco (10 min)
**Purpose:** Guarantee + fast-action bonus + penalty for waiting

Guarantee formula:
```
Garantia = Resultado + Prazo + Condicao + Consequencia

"Se em 90 dias voce nao tiver [RESULTADO],
mesmo aplicando [CONDICAO MINIMA],
eu [CONSEQUENCIA FAVORAVEL]"
```

### Block 7: Demonstracao Final (30 min)
**Purpose:** Q&A + live consultation + final close

Elements:
- Live Q&A
- Consultative demonstration
- Timer countdown
- Final urgency push
