## Purpose
Create a registration page that maximizes sign-ups for webinars, workshops, or events.

> "Registration is not the goal. Attendance is the goal. Your page should sell showing up, not just signing up." — Alex Hormozi

---

## Registration Page Structure

### Above the Fold

1. **Headline** — Event promise + date/time
2. **Subheadline** — What they'll learn/get
3. **Hero** — Presenter image or event visual
4. **Form** — Name, email, phone (optional)
5. **CTA Button** — "Reserve My Spot" / "Register Free"

### Below the Fold

6. **What You'll Learn** — 3-5 bullet points
7. **Who This Is For** — Avatar description
8. **About the Host** — Brief credibility
9. **Testimonials** — Past event results
10. **FAQ** — Common questions

---

## Key Elements

### Date/Time Display
Show clearly:
- Day of week
- Date
- Time with timezone
- "Add to calendar" option

### Social Proof
- "X people have registered"
- Past attendee testimonials
- Results from previous events

### Urgency Elements (Real Only)
- Limited seats (if true)
- Replay availability (be honest)
- Bonuses for live attendance

---

## Registration Form Optimization

### Minimum Fields
- Email (required)
- First name (required for personalization)

### Optional Fields
- Phone (if SMS reminders enabled)
- Company (if B2B targeting)

### Form Tips
- Single column
- Big input fields
- Prominent submit button
- Privacy note below form

---

## Headline Formulas

### Formula 1: Learn + Result + Time
```
"Learn How to [RESULT] in [TIMEFRAME]
FREE Live Training — [DATE]"
```

### Formula 2: Discover + Secret + Who
```
"Discover the [SECRET] That [WHO] Use to [RESULT]
Join Us [DATE] at [TIME]"
```

### Formula 3: Workshop + Promise
```
"Free [TOPIC] Workshop:
[BIG PROMISE] in [DURATION]"
```

---

## What You'll Learn Section

Format as clear bullet points:
```
In this free training, you'll discover:

✓ [Specific outcome 1]
✓ [Specific outcome 2]
✓ [Specific outcome 3]
✓ [Specific outcome 4]
✓ Plus: [Bonus element]
```

---

## Execution Steps

### Step 1: Define the Event
- Topic
- Date/time
- Duration
- Format (webinar, workshop, etc.)

### Step 2: Write Headline + Bullets
Focus on outcomes, not content.

### Step 3: Create Credibility Section
Brief host bio with relevant credentials.

### Step 4: Build the Form
Minimum fields, maximum clarity.

### Step 5: Add Social Proof
Past results, registrant count, testimonials.

### Step 6: Set Up Confirmation + Reminders
- Thank you page
- Confirmation email
- Reminder sequence (24h, 1h before)

---

## Post-Registration Flow

### Thank You Page
- Confirm registration
- Add to calendar link
- "Share with a friend" option
- "What to expect" preview

### Reminder Sequence
- Immediately: Confirmation + calendar invite
- 24 hours before: Reminder + agenda preview
- 1 hour before: Final reminder + link
- At start: "We're live!" notification
