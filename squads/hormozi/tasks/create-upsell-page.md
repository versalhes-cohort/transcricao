## Purpose
Create a post-purchase upsell page that increases average order value immediately after the first purchase.

> "The best time to sell someone is right after they just bought. They're in buying mode. Don't waste it." — Alex Hormozi

---

## Upsell Page Structure

### Element 1: Congratulations Bridge
- Acknowledge their purchase
- Reinforce they made a great decision
- Create positive momentum

### Element 2: The Enhancement Offer
- What it is (complementary to main purchase)
- Why it matters now
- How it accelerates/enhances their result

### Element 3: Exclusive Pricing
- Special one-time price
- Comparison to regular price
- Savings highlighted

### Element 4: Speed/Ease Promise
- Quick implementation
- No extra effort required
- Immediate benefit

### Element 5: Urgency Element
- One-time offer (won't see this again)
- Timer or countdown
- Clear deadline

### Element 6: Two-Button Close
- YES button (add to order)
- NO button (decline offer)

---

## Upsell Categories

### Type 1: More of the Same
- Bulk discount
- Extended access
- Additional quantity

### Type 2: Speed Enhancement
- Done-for-you version
- Fast-track option
- Priority access

### Type 3: Complementary Add-on
- Tool that helps use main product
- Resource that enhances results
- Community or support

### Type 4: Higher Tier
- Premium version
- VIP access
- Additional features

---

## Execution Steps

### Step 1: Identify the Gap
What do they need after purchasing that would help them succeed faster?

### Step 2: Write the Bridge
```
Congratulations! Your [main purchase] is confirmed.

But wait — there's something that could make this even better...
```

### Step 3: Present the Enhancement
- Clear benefit statement
- Connection to main purchase
- Specific outcome

### Step 4: Create Urgency
- This page only appears once
- Price goes up after
- Limited availability

### Step 5: Design the Buttons
```
[YES! Add This to My Order for $X]

[No thanks, I'll pass on this exclusive offer]
```
