# Story Quality Checklist

**Story:** {story_name}
**Framework:** {framework_used}
**Date:** {date}

---

## 1. Structure

- [ ] Clear beginning, middle, end
- [ ] Inciting incident defined
- [ ] Rising action builds tension
- [ ] Climax is satisfying
- [ ] Resolution provides closure

## 2. Character

- [ ] Protagonist clearly defined
- [ ] Character wants something (goal)
- [ ] Character faces obstacles
- [ ] Character transforms/learns
- [ ] Audience can relate/empathize

## 3. Conflict

- [ ] Central conflict established
- [ ] Stakes are clear and meaningful
- [ ] Tension escalates appropriately
- [ ] Obstacles feel genuine
- [ ] Resolution addresses conflict

## 4. Emotional Arc

- [ ] Opens with hook/engagement
- [ ] Emotional beats vary (not monotone)
- [ ] Key moments land emotionally
- [ ] Ending evokes intended emotion
- [ ] Audience cares about outcome

## 5. Message/Theme

- [ ] Core message is clear
- [ ] Theme emerges organically
- [ ] Story supports (not preaches) message
- [ ] Takeaway is memorable
- [ ] Call to action (if applicable) is clear

## 6. Framework Compliance

### StoryBrand (SB7)
- [ ] Character identified
- [ ] Problem articulated (external, internal, philosophical)
- [ ] Guide introduced
- [ ] Plan provided
- [ ] Call to action clear
- [ ] Success painted
- [ ] Failure avoided

### Hero's Journey
- [ ] Ordinary world established
- [ ] Call to adventure
- [ ] Crossing threshold
- [ ] Tests and allies
- [ ] Ordeal/transformation
- [ ] Return with elixir

### Story Circle
- [ ] You (comfort zone)
- [ ] Need (desire)
- [ ] Go (unfamiliar)
- [ ] Search (adapt)
- [ ] Find (get what wanted)
- [ ] Take (pay price)
- [ ] Return (back)
- [ ] Change (transformed)

## 7. Delivery

- [ ] Appropriate length
- [ ] Pacing feels right
- [ ] Language suits audience
- [ ] Sensory details included
- [ ] Dialogue (if any) sounds natural
