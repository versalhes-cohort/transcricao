# PV_PA_009 - The Paradox of Self-Validation (Split Generation & QA)

**Type:** Decision Heuristic
**Phase:** 3 (Executor Selection)
**Agent:** @hybridOps:process-architect
**Pattern:** HO-PP-003 (Quality Gate Pattern)

## Purpose

To prevent recursive hallucinations and blind algorithmic pride, an AI Agent that generates an output MUST NEVER be the executor authorized to validate it for final release. The role of 'Generator' and 'Validator' must be strictly cloven.

```
PRIMARY BRANCH:
  IF (an Agent executes a Generative task impacting an external stakeholder/system)

TERMINATION: No AI model assesses its own truthfulness for final delivery.
CONSTRAINT: NEVER trust the prompt "Act as a critical reviewer of your own text below".
```

## Failure Modes

### Hallucinatory Incest (The Coward)
- **Trigger:** Writing a prompt that says "Write the code, then act as QA and check if it's correct, then deploy."
- **Manifestation:** The Agent confidently generates an error, confidently states the error is perfect code, and collapses the build layer.
- **Detection:** Failures bypass the standard review steps masked by polite AI reassurance.
- **Recovery:** Hard halt the pipeline. Inject a separate AI Clone (with opposing defensive heuristics) or Human to evaluate the string uncolored by generation bias.
- **Prevention:** Split the prompts into different system calls managed by a workflow engine.

### Bureaucratic Friction (The Zealot)
- **Trigger:** Requiring two layers of Human QA for a simple data extraction Worker script that has never failed.
- **Manifestation:** Humans get bored, rubber-stamp approvals, slowing things down without adding safety.
- **Recovery:** If the generator is deterministic (Worker), standard error logs suffice. Cliven gates apply strictly to Stochastic (Agents).

## Behavioral Evidence

**Example:** Strategy Pitch Bot
- **Context:** An internal AI tool generated strategy proposals and automatically scored them.
- **Information Available:** The tool gave every single one of its proposals a 9.5/10 score.
- **Heuristic Application:** Invoked the Separation of Powers. Kept Agent A as the "Brainstormer". Built Clone B (The Asshole Reviewer) explicitly prompted to find 3 reasons the strategy would fail. 
- **Outcome:** The average score dropped to 6/10, forcing Agent A to regenerate until it passed objectively. Pipeline quality surged instantly.
